################################################################################################

Readme : e-commerce online store
         including all  e-commerce functionality and features.
		 template by Bootstrapious - https://bootstrapious.com

################################################################################################

---------------------
 pages included
---------------------

- Homepage
- Category listing 
- Product detail
- Shopping cart
- Checkout (4 steps)
- Contact page
- Customer account page
- Wishlist Page
- Registration Page
- 404 page

---------------------
 CSS
---------------------

- The theme stylesheet is css/style.default.css. 

---------------------
 JavaScript
---------------------

- Apart from the Bootstrap JS components, the majority of JS is located in /js/front.js, others in js folder and /includes/external_js_links.php.

---------------------
 PHP
--------------------

- Most php scripts are within the page files and more in /includes/ and in /admin/ root folder. Database Connection file is located in /admin/includes/db_config.php 
  and /admin/includes/class.user.php
